Jarvis Assistant (macOS)

Desktop AI assistant written in Java.

Features
- GUI chat
- macOS speech
- memory system
- internet tools
- Jarvis style responses

Requirements

Install Java JDK 17+
Install Maven

Build

chmod +x build.command
./build.command

Run

chmod +x run.command
./run.command

to Run it: 
download the zip file

After Downloading ZIP

Run once in terminal:

chmod +x build.command
chmod +x run.command

Then:

double-click build.command

Result:

dist/JarvisAssistant.jar

Run:

double-click run.command
